## 전략
### 전략 1: DFS
- DFS로 주어진 규칙대로 단어를 만들어보면서 목표 단어가 몇번째인지 찾는 방법

### 전략 2: product (🗒️)
- 주어진 단어 순서 규칙은 사전순 정렬 결과와 같음
- 따라서, product로 가능한 모든 단어를 만든 후에 정렬하여 위치를 찾는 방법

### 전략 3: 수학적 풀이 (🗒️, 🤖)
- 각 자리에서 문자가 바뀔 때마다 건너뛰게 되는 단어의 수
    - 5번째 자리: 다음 글자가 없으므로 자기 자신만 바뀜 -> ${1}$
    - 4번째 자리: 뒤 5번째 자리가 비거나 A, E, I, O, U 중 하나 -> ${1 + 5^1 = 6}$
        - 📝 보충) 뒤 5번째 자리가 빈다 -> 4번째 자리에서 문자열이 끝난다 -> 1을 더함
    - 3번째 자리: 뒤 4, 5번째 자리의 모든 경우의 수 -> ${1 + 5^1 + 5^2 = 31}$
    - 2번째 자리: 뒤 3, 4, 5번째 자리의 모든 경우의 수 -> ${1 + 5^1 + 5^2 + 5^3 = 156}$
    - 1번째 자리: 뒤 2, 3, 4, 5번째 자리의 모든 경우의 수 -> ${1 + 5^1 + 5^2 + 5^3 + 5^4 = 781}$
- 일반화하여, 가중치 (${W_n}$)를 공비가 5인 등비수열의 합의 공식으로 구할 수 있음
    - ${W_n = \sum_{k=0}^{n} 5^k = \frac{5^{n+1} - 1}{5 - 1}}$
- 단어의 각 자리를 순회하면서 문자의 인덱스 ${\times}$ 가중치 ${+}$ 1을 더함
    - `+1`은 해당 글자 자체가 나타나는 순서
    - 예) EIO의 경우:
        - 첫 번째 글자: A로 시작하는 모든 단어를 건너뛰고, E가 처음 나오는 순서
            - -> ${1 \times 781 + 1 = 782}$
        - 두 번째 글자: EA, EE로 시작하는 모든 단어를 건너뛰고, EI가 처음 나오는 순서
            - -> ${2 \times 156 + 1 = 313}$
        - 세 번째 글자: EIA, EIE, EII로 시작하는 모든 단어들을 건너뛰고, EIO가 처음 나오는 순서
            - -> ${3 \times 31 + 1 = 94}$
